# Pipelines

よく使う依頼を、人間が頼みやすい単位でまとめた入口です。

- 通常は pipeline 名を覚えるより、やりたいことをそのまま依頼すればよい。
- AI は依頼内容から近い pipeline を選び、必要な基本 skill を内部で読み分ける。
- 手法を固定したいときだけ、pipeline 名や skill 名を明示する。
- pipeline は固定手順ではなく、必要に応じて基本 skill を行き来しながら進めるための入口として扱う。

## 現在の pipelines

- `implementation`
- `code-review`
- `browser-analysis`
- `ux-performance-audit`
